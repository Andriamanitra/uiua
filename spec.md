# Operators and Precedence

These are all the uiua operators, order by increasing precedence

| Operator(s)         | Name(s)                  | Associativity |
| ------------------- | ------------------------ | ------------- |
| =, !=, <, <=, >, >= | Comparisons              | Left          |
| +, -                | Addition, Subtraction    | Left          |
| *, /                | Multiplication, Division | Left          |
| ...                 | Double Composition       | Left          |
| .                   | Composition              | Left          |
